# Ambient Sound

DMS plugin for playing ambient sounds for focus and relaxation.

![Screenshot](screenshot.png)

## Features

- Play multiple ambient sounds simultaneously
- Master volume control
- Grid layout for easy sound selection
- Loop sounds continuously

## Sounds Included

- Rain
- Fireplace
- Waves
- Wind
- Storm
- Birds
- City
- Coffee Shop
- Stream
- Summer Night

## Installation

The sounds are sourced from [Blanket](https://github.com/rafaelmardojai/blanket) under GPL-3.0 license.
